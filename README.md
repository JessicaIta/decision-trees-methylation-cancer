# Análise de dados na base de Metilação e Câncer de Mama com Decision Trees e AdaBoost

Este experimento utilizou parte da base disponível no portal do NCBI (Gene Expression Omnibus, código de acesso GSE57285) na análise dos dados de metilação e câncer de mama com o algoritmo Decision Trees e com o AdaBoost.

Link de acesso a base completa: https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE57285
